## 本项目是基于Vue-cli搭建是项目————学生资讯网站
#### 采用技术：采用vue进行编写

#### 图表采用：G2Plot框架技术
#### 地图采用谷歌地图api

- 安装：npm install
- 运行：npm run serve
- 打包：npm run build
